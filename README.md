- 다관절 로봇팔 제어용 스터디

1. description : 로봇의 형태 파일.
2. control : 로봇의 FollowJointTrajectory Action 서버.
3. config : MoveIt2 사용에 필요한 설정 파일 및 예제 launch 파일.
4. driver : JTC가 생성해낸 데이터를 어디서, 어디로 read write 할 것인지에 대한 정의
5. bringup : 전체 시스템 launch 파일
